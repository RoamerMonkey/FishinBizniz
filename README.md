A quick local asymmetric multiplayer game, "Fishing Bizniz", where a Fisher and a Hunter compete or collaborate to get the most amount of fish possible for a higher score.

Created by Nuuluz, Siso D., and Roamer Monkey for a project for a university game course using Godot 4.4. All artwork was drawn by Nuuluz. Siso and Roamer Monkey programmed the game. Roamer Monkey designed the majority of the game.

I used royalty-free music from Purple Planet and sound effects from Freesound. Generative AI was not used in the creation of this game.
